% --- Tagged equation environment (fix hyperref duplicate destinations) -----
% amsmath's \tag{...} replaces the visible equation number but rolls the
% equation counter back at \end{equation}, so multiple \tag-labelled
% equations all end up placing their hyperref anchor at the SAME
% counter value (e.g. equation.24), triggering pdfTeX's "destination
% with the same identifier has been already used, duplicate ignored"
% warnings and breaking \ref navigation to the later tagged eqs.
%
% Workaround: use equation* (unnumbered, no counter anchor) and manually
% set \@currentlabel so \ref prints the custom tag. \phantomsection
% creates a unique hyperref destination right before the equation so
% each \label binds to its own fresh anchor. Usage:
%     \begin{taggedeqn}{P}{P}
%         \hat{\boldsymbol{\theta}} = \arg\min ...
%     \end{taggedeqn}
% First arg is the visible tag text, second is the slug for eq:<slug>.
\makeatletter
\newenvironment{taggedeqn}[2]{%
    \gdef\@currentlabel{#1}%
    \phantomsection\label{eq:#2}%
    \begin{equation*}\tag*{(#1)}%
}{%
    \end{equation*}%
}
\makeatother

\begin{abstract}
Non-Newtonian video rheometry~\cite{hamamichi2023nonnewtonian}
estimates Herschel--Bulkley parameters $(\sigma_Y, \eta, n)$ of
complex fluids---pastes, chunky sauces, granular slurries---that
parallel-plate rheometers cannot handle, by matching dam-break flows
against physics-based simulation. Despite a sophisticated
identifiability-mitigation pipeline (loss-Hessian analysis,
Poiseuille-flow proxy, active setup selection, iterative multi-setup
CMA-ES), inversion remains hours-per-video because the material
point method must be evaluated hundreds of times per inverse,
foreclosing per-material use.

We compress the inverse to a median of twelve seconds per video on
one consumer GPU through three coupled components. A regime-aware
hierarchical Mixture-of-GP surrogate---$540$ sparse GP experts
routed by a two-level Bayesian Gaussian-mixture gate and aggregated
via a robust Bayesian Committee Machine---replaces the MPM forward
map with calibrated predictive distributions. An
identifiability-aware inverse warm-starts CMA-ES from the routed
expert's nearest training neighbor, anchors it with a data-honest
log-$\sigma_Y$ soft prior, and combines all available setups in a
single joint objective; an active two-setup selector fuses
Hessian-orthogonality with surrogate-spread and local-perturbation
candidates via a switch policy. A non-interactive ChArUco
calibration pipeline cuts per-capture human effort from $25$ to
$10$ minutes, making the end-to-end wall-clock field-deployable.

Evaluated against simulation-in-the-loop inversion on held-out
synthetic samples and real non-Newtonian fluids---including
head-to-head comparison on three real-video materials---our method
matches baseline accuracy at three orders of magnitude lower
latency. Seconds-scale inversion unlocks capabilities inaccessible
to prior work: \emph{(i)}~fresh parameter estimates for time-varying
fluids, tracked across repeated captures rather than frozen at a
single snapshot; \emph{(ii)}~practical characterization of materials
that parallel-plate rheometers cannot measure, extending rheometry
to chunky, granular, and large-inclusion samples;
\emph{(iii)}~a multi-observation joint protocol approaching
parallel-plate precision from phone videos alone, run as a routine
per-material step rather than a dedicated multi-hour sweep.
\end{abstract}
