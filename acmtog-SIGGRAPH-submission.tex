% --- Tagged equation environment (fix hyperref duplicate destinations) -----
% amsmath's \tag{...} replaces the visible equation number but rolls the
% equation counter back at \end{equation}, so multiple \tag-labelled
% equations all end up placing their hyperref anchor at the SAME
% counter value (e.g. equation.24), triggering pdfTeX's "destination
% with the same identifier has been already used, duplicate ignored"
% warnings and breaking \ref navigation to the later tagged eqs.
%
% Workaround: use equation* (unnumbered, no counter anchor) and manually
% set \@currentlabel so \ref prints the custom tag. \phantomsection
% creates a unique hyperref destination right before the equation so
% each \label binds to its own fresh anchor. Usage:
%     \begin{taggedeqn}{P}{P}
%         \hat{\boldsymbol{\theta}} = \arg\min ...
%     \end{taggedeqn}
% First arg is the visible tag text, second is the slug for eq:<slug>.
\makeatletter
\newenvironment{taggedeqn}[2]{%
    \gdef\@currentlabel{#1}%
    \phantomsection\label{eq:#2}%
    \begin{equation*}\tag*{(#1)}%
}{%
    \end{equation*}%
}
\makeatother

\begin{abstract}
We estimate the three Herschel--Bulkley parameters
(yield stress $\sigma_Y$, power-law index $n$, consistency parameter
$\eta$) of complex fluids---pastes, chunky sauces, granular
slurries---for which conventional rheometers cannot provide useful
measurements, by matching dam-break video footage against
physics-based simulation, following the protocol
of~\cite{hamamichi2023nonnewtonian}. The inverse is accurate but
slow: each estimation requires hundreds of in-loop
material-point-method evaluations and therefore takes around eight
hours of GPU time per setup, so the method serves as a research
tool rather than a routine measurement.

We compress this inverse from around eight hours to under a minute
per setup by replacing the in-loop simulator with a mixture of
Gaussian-process experts fitted to a corpus of MPM dam-break
simulations. We further replace the manual camera-calibration
workflow that precedes the inverse with an algorithmic two-stage
pipeline based on ChArUco markers and silhouette-edge refinement.

We validate our method by comparing against the simulation-in-the-loop
predecessor on real-video captures and against parallel-plate
rheometer measurements on materials the rheometers can handle, and
find that the recovered parameters are on par with the predecessor
at three orders of magnitude lower latency. The seconds-scale
inversion enables three uses inaccessible at hours-per-setup cost:
\emph{(i)}~tracking how a material's parameters evolve in real
time; \emph{(ii)}~characterizing many materials, batches, or
conditions within a single measurement session; and
\emph{(iii)}~iterative measurement workflows where each capture's
parameters guide the next adjustment.
\end{abstract}
